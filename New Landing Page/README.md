# SnapTagTrak Website

A modern, responsive website for SnapTagTrak - a receipt management and tracking solution.

## Features

- Responsive design that works on all devices
- Modern UI with smooth animations
- Optimized performance
- Cross-browser compatibility

## Project Structure

```
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Main stylesheet
├── js/
│   └── main.js         # JavaScript functionality
└── assets/
    └── images/         # Image assets
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - it's a static website

## Development

- The website uses vanilla HTML, CSS, and JavaScript
- No build tools or dependencies required
- CSS uses modern features like CSS Grid and Flexbox
- JavaScript includes smooth scrolling and intersection observer for animations

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
